# Neon Microstory Lab

Webová prezentace ke školnímu projektu předmětu ZMT. Obsahuje futuristicky laděnou stránku, která shrnuje námět, workflow, použitá média a vkládá výsledné ozvučené video (YouTube iframe).

## Struktura

- `index.html` – hlavní stránka s obsahem sekcí (myšlenka, workflow, media lab, video, proces).
- `styles/main.css` – neon/glassmorphism vzhled, grid layout, responzivní úpravy.
- `assets/` – místo pro snímky náhledů (nahraďte vlastními exporty).
- `docs/brief.md` – textové shrnutí ideového záměru.
- `docs/edit-log.md` – podrobný log úprav obrazových, video a audio stop.

## Rychlé spuštění

```
npx serve .
```

nebo jednoduše otevřete `index.html` v prohlížeči. V iframe nahraďte `VIDEO_ID` skutečným YouTube ID výsledného videa.
